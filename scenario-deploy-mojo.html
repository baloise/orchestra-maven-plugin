orchestra:scenario-deploy

Full name:

com.baloise.maven:orchestra-maven-plugin:0.7.3-SNAPSHOT:scenario-deploy

Description:

Deploys ( or redeploys ) a scenario ( PSC file ).

Example usage: 
mvn com.baloise.maven:orchestra-maven-plugin scenario-deploy -Duser=admin -Dpassword=*** -Dserver=https://orchestra.example.com:8019 -DpscFile=/tmp/my.psc -landscapeFile=/tmp/landscape.json
If you use a pom.xml feel free to omit pscFile

Attributes:

Required Parameters

Name Type Since Description
<artifactId> String 0.4.0 (no description)
Default value is: ${project.artifactId}.
User property is: artifactId.
<outputDirectory> File 0.4.0 the directory from where the PSC file will be read.
If you use the pscFile property, this will be ignored.
Default value is: ${project.build.directory}.
User property is: outputDir.
<password> String 0.4.0 (no description)
User property is: password.
<server> String 0.4.0 Full server URL with protocol and web service port.
Please note the web UI and web service do not run on the same port.
Example: https://orchestra.example.com:8019
User property is: server.
<user> String 0.4.0 (no description)
User property is: user.
<version> String 0.4.0 (no description)
Default value is: ${project.version}.
User property is: version.

Optional Parameters

Name Type Since Description
<comment> String 0.4.0 (no description)
User property is: comment.
<landscapeFile> String 0.4.0 Full path to the landscape file to deploy.
Each existing landscape entry will be overwritten.
If you don't give a property name ( see ee_mailto ) the default name VALUE will be used.
Property names are not case sensitive.
Arrays are joined with ", " (comma, blank) as separator.
{
"ee_openhr": {
"host": "myserver.example.com",
"port": "443"
},
"ee_mailto": [
"receiver1@example.com",
"second_receiver@example.com",
]
}
You can also put several landscapes into the same JSON and select them with
(on Microsoft Windows systems use ';' instead of ':') 
-DlandscapeFile=path/to/landscape.json:env/test

{"env": { "test" : { ... test landscape here ... }}}

User property is: landscapeFile.
<pscFile> File 0.4.0 full path to where is PSC file will be written.
Overwrites outputDir
User property is: pscFile.
<retryCount> int 0.5.2 (no description)
Default value is: 30.
User property is: retryCount.
<retryDelayMillies> long 0.5.2 (no description)
Default value is: 1000.
User property is: retryDelayMillies.
